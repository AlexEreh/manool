## Запуск

Рррр запускаем dev-сервер:

```bash

# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Открываем [http://localhost:3000](http://localhost:3000) с помощью браузера и наслаждаемся няяя~~~.
